Herramienta NSComment

La ubicada en el directorio /nscomment y subcarpetas. Esta herramienta está diseñada para ser una herramienta de gestión de comentarios de un post específico de Facebook, indicado por el usuario en la aplicación web.

El servidor no almacena ningún ningún dato obtenido a través de la API de Facebook, o de cualquier otro recurso externo generado por Facebook.

La aplicación web genera, por petición y manipulación directa del propietario de la cuenta de Facebook administrada por la aplicación, un reporte de los comentarios con datos de resumen: Fecha del comentario, nombre del usuario que comenta, correo electrónico principal del usuario que comenta y contenido del comentario (en cadena de texto). Este resumen se descargará en un archivo de Excel de la computadora del cliente (usuario de Facebook), a petición de él mismo.

Este paso solo puede ser realizado sólo por el usuario de Facebook haciendo uso de la aplicación web y para su uso personal. El servidor o el administrador del sitio web no conservan, obtienen o registran información alguna de la cuenta de Facebook en cuestión ni del post o los comentarios procesados, ni en ningun otro aspecto. El usuario es el único beneficiario del uso de la aplicación web.
